项目

个人项目 Janiva Programming Language

这是个做着玩的项目。

Janiva 是基于 GraalVM 的 Truffle 语言实现框架开发的 Json 模板语言,定位是 Json 语言的超集。

Janiva 相比 Json 最大的特点是它支持函数，包括：

  • 递归函数
  • 函数组合
  • Partial Application
  • 一些内置函数
