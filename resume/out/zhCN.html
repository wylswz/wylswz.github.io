个人项目 Janiva Programming Language

Janiva 是基于 GraalVM 的 Truffle 语言实现框架开发的 Json 模板语言,定位是 Json 语言的超集。可能的应用场景之一是 K8s 资源的动态渲染以及语言互操作性带来的可扩展的软件架构（类似 cue 语言之于 kubevela 的作用）

Janiva 相比 Json 最大的特点是它支持函数，包括

  • 递归函数
  • 函数组合
  • Partial Application
  • 一些内置函数
