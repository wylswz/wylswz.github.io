Reinforcement Learning

Reinforcement learning is used when the model of problem is unknown to us.

  • Q Learning:

    Q(s,a)=Q(s,a)+λ[r+γ⋅maxa′Q(s′,a′)−Q(s,a)]Q(s,a) = Q(s,a) + \lambda [r + \gamma\cdot max_{a'}Q(s',a') - Q(s,a)] Q(s,a)=Q(s,a)+λ[r+γ⋅maxa′​Q(s′,a′)−Q(s,a)]

  • SARSA:

    Q(s,a)=Q(s,a)+λ[r+γ⋅Q(s′,a′)−Q(s,a)]Q(s,a) = Q(s,a) + \lambda [r + \gamma \cdot Q(s',a') - Q(s,a)] Q(s,a)=Q(s,a)+λ[r+γ⋅Q(s′,a′)−Q(s,a)]

Q-Learning differs from SARSA that it has the assumption that the next state is taking optimal action, but in fact it does not necessarily (agent may need to explore, thus non-optimal action is taken). While SARSA uses the Q of next step the update the current Q, which means if the agent take non-optimal action at s′s's′, this will affect updating Q value at the current state. Therefore if ϵ\epsilonϵ-greedy is used, SARSA tend to train a more conservative policy instead of aggresive one.
